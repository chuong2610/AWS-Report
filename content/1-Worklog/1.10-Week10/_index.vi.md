### Mục tiêu tuần 10:

* Nắm vững cơ bản về serverless computing với AWS Lambda
* Xây dựng serverless backends hoàn chỉnh với Lambda, API Gateway và DynamoDB
* Triển khai authentication và authorization với Amazon Cognito
* Cấu hình custom domains và SSL certificates cho serverless APIs
* Thiết kế kiến trúc event-driven với S3, SQS và SNS
* Điều phối workflows phức tạp với AWS Step Functions
* Xây dựng real-time GraphQL APIs với AWS AppSync
* Giám sát và trace serverless applications với AWS X-Ray
* Tự động hóa serverless deployments với AWS SAM

### Kết quả đạt được tuần 10:

* Hiểu AWS là gì và nắm được các nhóm dịch vụ cơ bản: 
  * Compute
  * Storage
  * Networking 
  * Database
  * ...

* Đã tạo và cấu hình AWS Free Tier account thành công.

* Làm quen với AWS Management Console và biết cách tìm, truy cập, sử dụng dịch vụ từ giao diện web.

* Cài đặt và cấu hình AWS CLI trên máy tính bao gồm:
  * Access Key
  * Secret Key
  * Region mặc định
  * ...

* Sử dụng AWS CLI để thực hiện các thao tác cơ bản như:

  * Kiểm tra thông tin tài khoản & cấu hình
  * Lấy danh sách region
  * Xem dịch vụ EC2
  * Tạo và quản lý key pair
  * Kiểm tra thông tin dịch vụ đang chạy
  * ...

* Có khả năng kết nối giữa giao diện web và CLI để quản lý tài nguyên AWS song song.
