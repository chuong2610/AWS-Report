### Mục tiêu tuần 11:

* Nắm vững chiến lược hiện đại hóa ứng dụng từ monolith sang microservices
* Áp dụng Strangler Fig Pattern cho migration tăng dần
* Xây dựng microservices độc lập với database-per-service pattern
* Triển khai asynchronous microservices communication với messaging
* Xây dựng CI/CD pipelines hoàn chỉnh cho continuous delivery
* Chấp nhận văn hóa và thực hành DevOps với công cụ AWS
* Deploy ứng dụng đơn giản với AWS Elastic Beanstalk
* Xây dựng kiến trúc WordPress doanh nghiệp trên AWS

### Kết quả đạt được tuần 11:

* Hiểu AWS là gì và nắm được các nhóm dịch vụ cơ bản: 
  * Compute
  * Storage
  * Networking 
  * Database
  * ...

* Đã tạo và cấu hình AWS Free Tier account thành công.

* Làm quen với AWS Management Console và biết cách tìm, truy cập, sử dụng dịch vụ từ giao diện web.

* Cài đặt và cấu hình AWS CLI trên máy tính bao gồm:
  * Access Key
  * Secret Key
  * Region mặc định
  * ...

* Sử dụng AWS CLI để thực hiện các thao tác cơ bản như:

  * Kiểm tra thông tin tài khoản & cấu hình
  * Lấy danh sách region
  * Xem dịch vụ EC2
  * Tạo và quản lý key pair
  * Kiểm tra thông tin dịch vụ đang chạy
  * ...

* Có khả năng kết nối giữa giao diện web và CLI để quản lý tài nguyên AWS song song.
* ...
