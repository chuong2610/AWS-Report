This worklog documents my 12-week AWS learning journey during the internship program, spanning from **September 8, 2025** to **November 30, 2025**. Throughout this period, I systematically explored AWS services, built hands-on projects, and developed practical cloud engineering skills.

### Program Overview

The program was completed over **12 weeks** with structured weekly objectives covering fundamental to advanced AWS topics. Each week included practical labs, hands-on exercises, and real-world implementation scenarios.

### Weekly Learning Path

**Week 1:** [AWS Fundamentals & Core Services](1.1-week1/) - Introduction to cloud computing, AWS Global Infrastructure, IAM basics, VPC networking, and EC2 instance management

**Week 2:** [Compute & Storage Deep Dive](1.2-week2/) - Advanced EC2 configurations, EBS volumes, AWS Cloud9 IDE, Amazon S3 storage, and AWS Lightsail for simplified deployments

**Week 3:** [Managed Databases & High Availability](1.3-week3/) - Amazon RDS for relational databases, DynamoDB for NoSQL, ElastiCache for in-memory caching, and designing highly available architectures

**Week 4:** [Scalability & Content Delivery](1.4-week4/) - Auto Scaling Groups, CloudWatch monitoring and alarms, Route 53 DNS management, and CloudFront CDN for global content delivery

**Week 5:** [Infrastructure as Code & Automation](1.5-week5/) - AWS Systems Manager for operational tasks, CloudFormation for infrastructure provisioning, AWS CDK for programmatic infrastructure, and ABAC authorization

**Week 6:** [Security & Identity Management](1.6-week6/) - AWS WAF for web application protection, KMS for encryption key management, Secrets Manager, Amazon GuardDuty threat detection, and Amazon Cognito for user authentication

**Week 7:** [Migration & Messaging Services](1.7-week7/) - VM migration strategies, AWS Database Migration Service, disaster recovery planning, Amazon SQS for queuing, and SNS for notifications

**Week 8:** [Cost Optimization & Networking](1.8-week8/) - AWS cost management best practices, FinOps principles, AWS Transit Gateway for network connectivity, and resource optimization strategies

**Week 9:** [Containerization & Orchestration](1.9-week9/) - Docker containerization fundamentals, Amazon ECS for container management, AWS Fargate for serverless containers, and Amazon EKS for Kubernetes

**Week 10:** [Serverless Architecture](1.10-week10/) - AWS Lambda functions, serverless computing patterns, Amazon API Gateway for RESTful APIs, and AWS Step Functions for workflow orchestration

**Week 11:** [Microservices & CI/CD](1.11-week11/) - Microservices architecture patterns, continuous integration and deployment pipelines, DevOps practices, and AWS Elastic Beanstalk for application deployment

**Week 12:** [Data Analytics & Machine Learning](1.12-week12/) - Data lake architecture with AWS Lake Formation, AWS Glue for ETL, Amazon Athena for SQL queries, and Amazon SageMaker for machine learning model development
